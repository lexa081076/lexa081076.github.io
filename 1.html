Десктопная версия (1920x1080px)

Здесь будет представлен макет десктопной версии сайта

Использование модульной сетки

В десктопной версии используется 12-колоночная сетка с отступами по 20px. Контент центрирован с максимальной шириной 1200px.

Применение стилей

Для десктопной версии используются полноразмерные компоненты с детализацией всех элементов интерфейса.

Принципы адаптивности

Десктопная версия оптимизирована для экранов шириной от 1200px, с использованием всех преимуществ большого экрана.

Мобильная версия (375x667px)

Здесь будет представлен макет мобильной версии сайта

Использование модульной сетки

В мобильной версии используется 6-колоночная сетка с уменьшенными отступами. Навигация преобразуется в гамбургер-меню.

Применение стилей

Для мобильной версии используются упрощенные компоненты с увеличенными областями касания и адаптированной типографикой.

Принципы адаптивности

Мобильная версия использует медиа-запросы для адаптации контента, изменяет расположение элементов и размеры шрифтов для удобства использования на маленьких экранах.
